# pnlk_PDF_merge

Переработал код в файле с объединением PDF файлов с панелями. 
Для объединения файлов была проверка чтобы последние числа были в диапазоне [1-100] и обязательно начинались с 1.

 https://chat.deepseek.com/a/chat/s/cae19327-79e6-4501-87bb-d58a0db9b159


 Основные изменения:
1) Добавлена функция validate_numbered_files(files), которая проверяет:
 Что все файлы имеют номера в диапазоне [1-100]
 Что первый номер равен 1
 (Опционально) Что номера идут последовательно без пропусков
2) Изменена логика в process_pdf_files():
 Группы с несколькими файлами сначала проходят валидацию
 Если валидация не пройдена, файлы копируются как одиночные
 Добавлены информативные сообщения о причинах отказа от объединения
3) Уточнены сообщения для пользователя:
 В итоговом отчете добавлена информация о требованиях к номерам файлов
Теперь программа будет объединять только те группы файлов, у которых номера находятся строго в диапазоне от 1 до 100 и начинаются с 1. Все остальные файлы будут скопированы как одиночные.
